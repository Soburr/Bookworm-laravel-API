## Bookworm Laravel API

## General Information
A simple Bookstore API built and tested with [Postman](https://postman.com) 

## Technologies Used
- LARAVEL PASSPORT
- LARAVEL V 10.2.5
- PHP V 8.1

## Set Up

```
git clone https://github.com/Soburr/Bookworm-Laravel-APi.git 
composer install
php artisan migrate

```
Then

```
npm install
npm run dev
```
Run the server

```
php artisan serve
```
