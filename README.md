## BOOKWORM LARAVEL API

## Technologies Used
- LARAVEL PASSPORT
- LARAVEL V 10.2.5
- PHP V 8.1

## Set Up
git clone https://github.com/Soburr/Bookworm-Laravel-APi.git 
composer install
php artisan migrate
Then
npm install
npm run dev
Run the serve
php artisan serve
Some basic Git commands are:
```
git status
git add
git commit
```
